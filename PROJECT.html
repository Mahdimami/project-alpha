Introduction to CSS

Cascading Style Sheets (CSS) is a stylesheet language used for describing the look and formatting of a document written in HTML. It can be used to add color, font, spacing, and other styles to web pages. CSS allows you to separate the presentation and content of a web page, making it easier to maintain and update.

There are three ways to add CSS styles to a web page: inline styles, internal stylesheets, and external stylesheets. Inline styles are added directly to HTML elements using the "style" attribute. Internal stylesheets are placed in the "head" section of the HTML document, and external stylesheets are linked to the HTML document using the "link" element.

CSS has a wide range of selectors and properties that can be used to style HTML elements. Some of the most commonly used CSS properties include: font-size, color, background-color, border, margin, and padding. By using CSS, you can create complex and sophisticated web designs with ease.
